#### 步驟 3️⃣：開始演示！

1. 訪問 https://claude.ai/code
2. 連接你的 GitHub 帳號
3. 選擇 `lewis100challenges/claude-code-demo`
4. 使用這個 Prompt 開始：

```
為這個用戶註冊系統添加完整的輸入驗證功能：

1. 前端驗證（signup.ts）：
   - 用戶名：3-20 個字符，只能包含字母、數字和下劃線
   - Email：必須是有效的 email 格式
   - 密碼：至少 8 個字符

2. 後端驗證（service.go）：
   - 添加相同的驗證邏輯
   - 確保所有必填字段都存在
   - 返回清晰的錯誤訊息

3. 同時在前端和後端都添加密碼強度驗證：
   - 必須包含至少一個大寫字母
   - 必須包含至少一個小寫字母
   - 必須包含至少一個數字
```
